- For self-hosted builds, please download from [Github Release](https://github.com/dogu-team/dogu/releases).

- Please run the exe installation file and agree to allow additional
  network access.

:::caution

Installation may fail due to conflicts with security programs. If you suspect a conflict, you can add the $HOME/.dogu folder as a security exception folder.
[For Microsoft Defender](https://support.microsoft.com/en-us/windows/add-an-exclusion-to-windows-security-811816c0-4dfd-af4a-47e4-c301afe13b26)

:::
