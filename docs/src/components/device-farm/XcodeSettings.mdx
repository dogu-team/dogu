<summary mdxType="summary">Xcode Settings</summary>

1. Open App Store

   `command + space` to open spotlight search, then type `app store` to open app store.

2. Login on App Store

   Login with your Apple ID.

3. Install Xcode

   Search `xcode` in app store, then install it.

4. Open Xcode

   Open Xcode, then accept the license agreement.

5. Login on Xcode

   Login with your Apple ID.

6. Check xcode-select

   type the following command in terminal:

   ```bash
   xcode-select -p
   ```

   If the output is `/Applications/Xcode.app/Contents/Developer`, then _you can skip the next step_.

7. Set xcode-select

   type the following command in terminal:

   ```bash
    sudo xcode-select -s /Applications/Xcode.app/Contents/Developer
   ```
