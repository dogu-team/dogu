- Self-hosted 빌드의 경우에는 [Github Release](https://github.com/dogu-team/dogu/releases)에서 다운로드 받아주세요.

- exe 설치파일을 실행하고, 추가 네트워크 액세스 허용을 동의해주세요.

:::caution

보안프로그램과 충돌로 인해 설치가 실패할 수도 있습니다. 만약 충돌이 의심된다면 $HOME/.dogu 폴더를 보안 예외 폴더로 추가하는 방법이 있습니다.
[Microsoft Defender의 경우](https://support.microsoft.com/en-us/windows/add-an-exclusion-to-windows-security-811816c0-4dfd-af4a-47e4-c301afe13b26)

:::
