# Toolkit 소개

Toolkit은 게임 제어, 디바이스 제어기능을 제공하는 패키지입니다.

`Gamium`은 게임플레이를 자동화할 수 있는 SDK입니다. `Gamium` 통해 게임유저의 행동을 스크립트화할 수 있는 기능을 제공합니다.
`Gamium`은 게임엔진 내부 요소들은 제어가 가능하지만, Native 디바이스 제어가 불가능합니다. 예를 들어 Unity에 Gamium Engine을 적용할 경우 Unity에서 제작한 UI요소는 제어가 가능합니다. 하지만 Android의 권한 허용창을 제어할 수 없습니다.

하지만 설치, 실행, 테스트 라는 하나의 프로세스 중에서는 Native 디바이스 제어를 필요로 하거나 필요한 경우가 생기는 것이 빈번합니다.
이런 경우를 해결하기 위해 Dogu에서는 Toolkit을 개발했습니다.

## 시작하기

Toolkit은 Gamium과 Device 제어 기능을 포함하고 있는 패키지 입니다. 이 말은 Toolkit하나로 특정 디바이스에 앱 설치, 실행, 게임 UI제어, 디바이스 제어가 가능하다는 의미입니다.
Toolkit은 [시작하기](quick-start) 문서를 통해 시작할 수 있습니다.
