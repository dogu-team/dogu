API 토큰을 통해 Dogu API를 외부에서 사용할 수 있습니다

## API 토큰

:::danger

API 토큰은 외부에 노출되지 않도록 주의해주세요.

:::

API 토큰은 Dogu를 통해 외부에서 Dogu API를 사용할 수 있도록 해주는 키입니다.  
Dogu에서 조직을 만들 때 자동으로 생성되며, 조직 내 설정 페이지에서 확인할 수 있습니다.

:::info

토큰 관리에 대한 내용은 [조직 설정](/management/organization/settings) 문서를 참고하세요.

:::
