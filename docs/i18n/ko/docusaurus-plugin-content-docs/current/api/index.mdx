엑세스 토큰을 통해 Dogu API를 외부에서 사용할 수 있습니다

# Access Token

:::danger

엑세스 토큰은 외부에 노출되지 않도록 주의해주세요.

:::

### 조직 엑세스 토큰

Dogu에서 조직을 만들 때 자동으로 생성되며, 조직 내 설정 페이지에서 확인할 수 있습니다.

:::info
조직 엑세스 토큰은 조직의 `Admin` 권한을 가집니다.  
조직 엑세스 토큰 관리에 대한 내용은 [조직 엑세스 토큰](/management/organization/settings#access-token) 문서를 참고하세요.
:::

### 프로젝트 엑세스 토큰

Dogu에서 프로젝트를 만들 때 자동으로 생성되며, 프로젝트 내 설정 페이지에서 확인할 수 있습니다.

:::info
프로젝트 엑세스 토큰은 프로젝트의 `Admin` 권한을 가집니다.  
프로젝트 엑세스 토큰 관리에 대한 내용은 [프로젝트 엑세스 토큰](/management/project/settings#access-token) 문서를 참고하세요.
:::

### 개인 엑세스 토큰

Dogu에서 계정을 만들 때 자동으로 생성되며, 계정의 설정 페이지에서 확인할 수 있습니다.

:::info
개인 엑세스 토큰은 해당 계정의 권한을 가집니다.  
개인 엑세스 토큰 관리는 계정 설정 페이지에서 확인할 수 있습니다.
:::
