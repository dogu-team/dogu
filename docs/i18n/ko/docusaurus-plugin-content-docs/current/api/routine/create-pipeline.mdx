```
POST /v1/projects/:projectId/routines/:routineId/pipelines
```

### Query Parameters

| Attribute   | type   | required | description                                                                                                                            |
| ----------- | ------ | -------- | -------------------------------------------------------------------------------------------------------------------------------------- |
| `projectId` | string | true     | 프로젝트의 ID입니다. 프로젝트 설정 페이지에서 쉽게 찾을 수 있습니다. [프로젝트 설정](/management/project/settings) 문서를 참고해주세요 |
| `routineId` | string | true     | 루틴 ID입니다. [루틴 ID 확인](/management/project/routine#check-routine-id)을 참고해주세요.                                            |

### Headers

| Attribute       | type   | required | description                                                                                                 |
| --------------- | ------ | -------- | ----------------------------------------------------------------------------------------------------------- |
| `Authorization` | string | true     | Authorization: Bearer ${ACCESS_TOKEN}.<br />ACCESS_TOKEN은 [엑세스 토큰](/api/access-token)을 참고해주세요. |

### Example

```
curl -v -X POST "http://localhost:4000/v1/projects/${projectId}/routines/${routineId}/pipelines" \
-H "Authorization: Bearer ${ACCESS_TOKEN}" \
-H "Content-Type: application/json"
```
