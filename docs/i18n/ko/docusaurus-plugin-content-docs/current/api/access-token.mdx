## Access Token

Dogu API를 외부에서 사용하기 위한 정보입니다.

:::danger

Access Token은 외부에 노출되지 않도록 주의해주세요.

:::

## Organization Access Token

Dogu에서 조직을 만들 때 자동으로 생성되며, 조직 내 설정 페이지에서 확인할 수 있습니다.

:::info
Organization Access Token은 조직의 `Admin` 권한을 가집니다.  
Organization Access Token 관리에 대한 내용은 [조직 설정](/management/organization/settings) 문서를 참고하세요.
:::

## Project Access Token

Dogu에서 프로젝트를 만들 때 자동으로 생성되며, 프로젝트 내 설정 페이지에서 확인할 수 있습니다.

:::info
Project Access Token은 프로젝트의 `Admin` 권한을 가집니다.  
Project Access Token 관리에 대한 내용은 [프로젝트 설정](/management/project/settings) 문서를 참고하세요.
:::

## Personal Access Token

Dogu에서 계정을 만들 때 자동으로 생성되며, 계정의 설정 페이지에서 확인할 수 있습니다.

:::info
Personal Access Token은 해당 계정의 권한을 가집니다.
Personal Access Token 관리는 계정 설정 페이지에서 확인할 수 있습니다.
:::
