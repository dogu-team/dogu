해당 문서는 테스트 스크립트를 관리하는 방법에 대해 설명합니다.

## Dogu Gitlab

Dogu는 테스트 스크립트를 관리할 수 있는 자체적으로 [Gitlab](https://gitlab.dogutech.io)을 제공하고 있습니다.  
회원가입 시 자동으로 Dogu Gitlab에 가입되며, 해당 Gitlab 서비스는 Dogu 서비스와 연동됩니다.

:::info
Dogu Gitlab의 아이디 및 비밀번호는 Dogu 서비스의 아이디 및 비밀번호와 동일합니다.
:::
