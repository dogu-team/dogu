테스트 스크립트와 루틴을 생성하고 루틴을 실행하면 테스트 리포트를 제공합니다. 테스트 리포트는 디바이스 작업을 클릭해 볼 수 있습니다.

1. 프로젝트 페이지에서 "루틴" 탭을 클릭합니다.
2. 결과를 볼 파이프라인을 클릭합니다.
   <img
     src="/ko/img/routine/running-pipeline.png"
     style={{ width: 600, display: 'block' }}
   />
3. 왼쪽 사이드바에서 작업과 디바이스 작업을 선택합니다.
   <img
     src="/ko/img/routine/pipeline-live-sidebar.png"
     style={{ width: 600, display: 'block' }}
   />

테스트 결과는 디바이스 작업 또는 스텝에 따라 조금씩 다릅니다.

## 디바이스 작업

디바이스 작업에서는 다음과 같은 정보를 확인할 수 있습니다.

### 녹화 영상

:::info

녹화 영상은 루틴을 작성할 때 옵션을 활성화 하고, 디바이스 작업이 끝난 후에 보입니다.

루틴 작성 방법은 [루틴 편집](/routine/routines/editor) 또는 [루틴 YAML 문법](/routine/routines/syntax)을 참고하세요.

:::

디바이스 작업에서 녹화된 영상을 확인할 수 있습니다. 스텝 또는 그 하위 테스트 단위별 버튼을 클릭해 영상 시간을 조정할 수 있습니다.

<img
  src="/ko/img/routine/device-job-record.png"
  style={{ width: 600, display: 'block' }}
/>

### 디바이스 프로파일 정보

디바이스의 CPU 사용량, 메모리 사용량, 게임 FPS 그래프를 확인할 수 있습니다.

<img
  src="/ko/img/routine/device-job-device-profile.png"
  style={{ width: 600, display: 'block' }}
/>

1. CPU 사용량
   - CPU 사용량은 현재 디바이스의 `user` CPU 사용량, `system` CPU 사용량, 현재 디바이스가 실행중인 프로세스의 CPU 사용량을 표시합니다.
2. 메모리 사용량
   - 메모리 사용량은 현재 디바이스의 전체 메모리, 사용중인 메모리, 현재 디바이스가 실행중인 프로세스의 메모리 사용량을 표시합니다.
3. 게임 FPS
   - 게임 FPS는 현재 디바이스에서 실행중인 게임의 FPS를 표시합니다.

:::info FPS

게임 FPS 정보를 받아오기 위해서는 게임 프로젝트에 Gamium Engine SDK가 포함되어있어야 합니다. 자세한 내용은 [Gamium Engine SDK](https://gamium.dogutech.io/docs/engine/unity/project-configuration)를 참고하세요.

:::

### 라이브 로그

:::info

디바이스 작업이 실행중일 때 라이브 로그가 표기됩니다. 디바이스 작업이 끝난 후에는 스텝에서 로그를 확인해 주세요.

:::

라이브 로그에서는 디바이스 로그, 테스트 스크립트 로그를 확인할 수 있습니다.

<img
  src="/ko/img/routine/live-log.png"
  style={{ width: 600, display: 'block' }}
/>

### 스텝 목록

디바이스 작업에서 실행된 스텝 목록을 확인할 수 있습니다. 스텝은 루틴에서 job 내부에 정의한 스텝입니다.

<img
  src="/ko/img/routine/device-job-step-list.png"
  style={{ width: 600, display: 'block' }}
/>

## 스텝

스텝에서는 다음과 같은 정보를 확인할 수 있습니다.

### 테스트

테스트 스크립트를 수행하는 스텝의 경우 확인할 수 있습니다. 수행하는 테스트 목록이 표시됩니다.

<img
  src="/ko/img/routine/device-job-step-list.png"
  style={{ width: 600, display: 'block' }}
/>

### 로그

스텝이 종료되면 로그를 확인할 수 있습니다. 디바이스 작업과 동일하게 디바이스 로그와 테스트 스크립트 로그를 확인할 수 있습니다.

### 디바이스 프로파일

스텝이 종료되면 디바이스 프로파일 그래프를 확인할 수 있습니다. 디바이스 작업과 동일하게 CPU 사용량, 메모리 사용량, 게임 FPS 그래프를 확인할 수 있습니다.

## 테스트

테스트는 테스트 스크립트에서 정의한 `Job`과 `Test` 목록이 표시됩니다.

테스트에서는 아래와 같은 정보를 확인할 수 있습니다.

:::info

`Job`애서는 테스트 스크립트에서 정의한 `Test` 목록이 나열됩니다.

:::

:::caution

정의한 `Test`의 실행 시간이 짧은 경우, 정보가 표기되지 않을 수 있습니다.

:::

### 로그
