# Jest

Dogu는 jest 통합을 지원합니다.  
jest의 테스트 결과를 [**@dogu-tech/jest-environment**](https://www.npmjs.com/package/@dogu-tech/jest-environment) 패키지를 사용해 아래 화면처럼 루틴의 테스트 결과를 확인할 수 있습니다.

![@dogu-tech/jest-environment](/img/testing-framework-support/jest.png)

## 요구사항

- Node.js 16.0.0 이상

## 설정

- `@dogu-tech/jest-environment` 패키지를 프로젝트에 설치합니다.

```shell
# using npm
npm install --save-dev @dogu-tech/jest-environment

# using yarn
yarn add --dev @dogu-tech/jest-environment

# using pnpm
pnpm add --save-dev @dogu-tech/jest-environment
```

- `@dogu-tech/jest-environment`를 jest 설정에 추가합니다.

```javascript
// jest.config.js

/** @type {import('jest').Config} */
module.exports = {
  testEnvironment: '@dogu-tech/jest-environment',
  // ... 다른 jest 설정들
};
```

:::info jest 설정

자세한 정보는 [공식 문서](https://jestjs.io/docs/configuration)를 참고해주세요.

:::
