# dogu-actions/run-test

---

프로젝트 레포지토리의 테스트 스크립트를 실행합니다.

#### 소스 코드

[깃허브 링크](https://github.com/dogu-actions/run-test)

#### Usage

```yaml
# 작업 정의...
steps:
  - name: Run Test
    uses: dogu-actions/run-test
    with:
      script: test/samples/dogurpgsample.test.ts
```

:::caution

연동된 Git repository에 테스트 스크립트가 존재해야 합니다.

:::

## Arguments

### script

<code
  style={{ position: 'relative', top: '-20px', fontSize: '90%', color: 'red' }}
  children="required"
/>

테스트 스크립트 파일의 경로를 입력합니다.

```yaml
# test/samples/dogurpgsample.test.ts 파일을 실행합니다.
```
