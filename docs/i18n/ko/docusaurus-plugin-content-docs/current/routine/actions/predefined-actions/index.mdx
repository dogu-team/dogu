# 미리 정의된 액션

`uses` 키워드를 사용하여 미리 정의된 액션을 사용할 수 있습니다.

:::info
[루틴](/routine/routines) 실행시 Github 레포지토리에서 [호스트](/device-farm#host-and-agent)에 다운로드 및 실행됩니다.
:::

#### Usage

```yaml
steps:
  - name: prepare test
    uses: dogu-actions/prepare
    with:
      appVersion:
        android: 1.2.3
  - name:
    uses: dogu-actions/run-test
    with:
      script: test/samples/dogurpgsample.test.ts
```
