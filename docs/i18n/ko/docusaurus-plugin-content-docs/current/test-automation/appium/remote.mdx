## Remote Appium options

Appium remote 테스트를 실행할 경우에 옵션을 설명합니다. Appium remote 테스트를 실행하는 방법은 [Tutorial](/get-started/tutorials/test-automation/remote-testing/appium)을 참고해주세요.

## Dogu options

Webdriver에 기존 Capabilities를 Dogu에 맞게 확장한 [Extension capabilities](https://www.w3.org/TR/webdriver/#dfn-extension-capability) 입니다. 아래 key들은 `dogu:options` 라는 namespace 안에 설정가능한 필드들 입니다.

| key              | required | description                                                                                                                                               |
| ---------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `token`          | true     | [Access Token](/api)문서를 참고하여 Access Token을 획득하고 입력해주세요                                                                                  |
| `organizationId` | true     | 조직의 ID입니다. 프로젝트 설정 페이지에서 쉽게 찾을 수 있습니다. [프로젝트 설정](/management/project/settings) 문서를 참고하세요                          |
| `projectId`      | true     | 프로젝트의 ID입니다. 프로젝트 설정 페이지에서 쉽게 찾을 수 있습니다. [프로젝트 설정](/management/project/settings) 문서를 참고하세요                      |
| `runsOn`         | true     | 테스트를 실행할 타겟을 설정하는데 사용됩니다. 특정 디바이스의 이름 혹은 태그를 입력해주세요. 여러개를 설정할 경우 조건을 모두 만족하는 타겟이 설정됩니다. |
| `appVersion`     | false    | Project에 업로드된 앱 중 테스트하고자 하는앱의 버전입니다. [앱 관리](/management/project/app) 페이지를 참고하여 원하는 앱을 업로드해주세요                |
| `browserName`    | false    | 브라우저 테스트시에 사용됩니다. (ex. chrome)                                                                                                              |
| `browserVersion` | false    | 브라우저 테스트시에 사용됩니다. (ex. 91.0.4472.114)                                                                                                       |

## Overriden Capabilities

Dogu Server에서 자동으로 설정하는 Capabilities입니다.

| key                        | description                                                              | default                                          |
| -------------------------- | ------------------------------------------------------------------------ | ------------------------------------------------ |
| `platformName`             | `platformName`은 Dogu server가 Device 정보를 통해 자동으로 덮어씁니다.   | android: `android`<br></br> ios: `ios`           |
| `browserName`              | 만약 `dogu:options.browserName`이 있다면, 덮어씁니다.                    | {browserName}                                    |
| `browserVersion`           | 만약 `dogu:options.browserVersion`이 있다면, 덮어씁니다.                 | {browserVersion}                                 |
| `appium:automationName`    | Dogu server가 Device 정보를 통해 자동으로 덮어씁니다.                    | android: `uiautomator2`<br></br> ios: `xcuitest` |
| `appium:deviceName`        | 디바이스 serial 또는 udid                                                | android: {serial}<br></br> ios: {udid}           |
| `appium:udid`              | 디바이스 serial 또는 udid                                                | android: {serial}<br></br> ios: {udid}           |
| `appium:mjpegServerPort`   | host machine's free random port                                          | {유휴 임의 포트}                                 |
| `appium:app`               | 만약 `dogu:options.appVersion`이 있다면 다운로드한 앱 경로로 설정합니다. | {다운로드한 앱 경로}                             |
| `appium:systemPort`        | **ANDROID ONLY** 호스트 장비의 유휴 임의 포트                            | {유휴 임의 포트}                                 |
| `appium:chromedriverPort`  | **ANDROID ONLY** 호스트 장비의 유휴 임의 포트                            | {유휴 임의 포트}                                 |
| `appium:webDriverAgentUrl` | **IOS ONLY** 디바이스 WebdriverAgent 서버로 forward된 호스트 포트        | http://127.0.0.1:{유휴 임의 포트}                |
| `appium:showXcodeLog`      | **IOS ONLY** Xcode 로그를 Dogu Agent로 출력합니다.                       | true                                             |
| `appium:derivedDataPath`   | **IOS ONLY** WebDriverAgent의 빌드 경로                                  | {운영체제 임시 경로}                             |
