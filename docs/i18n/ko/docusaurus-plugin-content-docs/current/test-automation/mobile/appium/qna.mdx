:::note
self-hosted라면?

- https://api.dogutech.io/remote/wd/hub 로 적혀있는 url을 self-hosted dogu의 url로 변경해주세요.

:::

:::note
iOS를 테스트하고 싶다면?

- iOS 는 앱 재서명을 해야해서 아직 샘플앱을 제공하지 않습니다. 하지만 iOS도 내부 앱이 기기에 설치가능하다면 테스트할 수 있습니다. capabilities에 appium:automationName값을 xcuitest로 설정하고 platformName과 tag를 ios로 변경하여 테스트를 진행해주세요.

:::
