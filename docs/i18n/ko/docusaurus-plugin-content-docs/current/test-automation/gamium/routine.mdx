[Gamium](https://gamium.dogutech.io/)은 게임 테스트 자동화를 위한 라이브러리입니다. Dogu DeviceFarm에서는 Gamium을 Appium과 연동하여 테스트를 진행할 수 있습니다. Gamium routine 테스트를 진행하는 방법은 [Tutorial](/get-started/tutorials/test-automation/routine-testing/gamium)을 참고해주세요.
