Dogu는 Selnium, Appium, Gamium, Playwright 등의 테스트 자동화 도구를 지원합니다. 이 도구들과 연동하여 테스트 자동화를 수행할 수 있습니다.
Remote test를 통해 간단히 하나의 디바이스에서 테스트를 진행할 수 있고, Routine을 통해 멀티디바이스에 대한 테스트를 진행할 수 있습니다. 사용하고 있거나 사용을 희망하는 Testing Framework와 Dogu를 연동하여 테스트 자동화를 진행하고 콘솔에서 결과를 한 눈에 확인해보세요.

아직 Dogu를 사용해보지 않으셨다면 [Toturial](/get-started/tutorials)를 통해 Dogu를 사용해보세요.
이 문서에서는 각 Testing Framework연동시 추가 설정들에 대해 설명합니다.
