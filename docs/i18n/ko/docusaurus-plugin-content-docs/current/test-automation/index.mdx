Dogu Agent에는 Selenium Grid, Appium Server 등이 포함돼 있어 포함된 서버를 통해 디바이스 팜에 존재하는 디바이스를 대상으로 테스트를 원격으로 실행할 수 있습니다.

## Selenium Grid

Selenium Grid는 브라우저를 원격으로 제어할 수 있도록 도와주는 도구입니다. Selenium Grid 옵션에 대해 더 알아보려면 [Selenium](/test-automation/selenium)을 참고하세요.

## Appium Server

Appium Server는 앱을 원격으로 제어할 수 있도록 도와주는 도구입니다. Appium Server 옵션에 대해 더 알아보려면 [Appium](/test-automation/appium)을 참고하세요.
