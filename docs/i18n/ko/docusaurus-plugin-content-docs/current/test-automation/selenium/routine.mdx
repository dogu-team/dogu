## Routine Selenium options

루틴으로 브라우저 테스트를 실행할 때의 세부 사항을 설명합니다. 루틴을 사용하여 Selenium 테스트를 실행하는 방법은 [튜토리얼](/get-started/tutorials/test-automation/routine-testing/selenium)을 참조하세요.

## Autoinstall browsers

dogu-device-client를 사용하여 verifyBrowserAndDriver()를 호출하면 테스트용 브라우저가 자동으로 설치됩니다. dogu-device-client 사용 예시는 [dogu-routine-examples](https://github.com/dogu-team/dogu-routine-examples)에서 확인할 수 있습니다.
