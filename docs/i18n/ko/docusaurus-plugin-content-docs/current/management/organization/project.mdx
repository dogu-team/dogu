프로젝트는 **하나의 소프트웨어(모바일 앱, 웹, 게임)**에 대한 테스트 스크립트, 루틴, 디바이스 관리, 프로젝트 멤버 등을 관리할 수 있습니다.

한 조직 내에서 여러 프로젝트를 생성할 수 있습니다.

프로젝트에서는 아래와 같은 기능을 제공합니다.

- 프로젝트 멤버 또는 팀을 관리합니다.
- 프로젝트에서 사용할 디바이스를 관리합니다.
- 테스트 스크립트를 작성 및 관리합니다.
- 테스트를 위한 루틴을 만들고, 할당된 디바이스를 통해 자동화 테스트를 수행합니다.

:::tip 자세히 알아보기

프로젝트 내 기능에 대한 자세한 사항은 [프로젝트 문서](../project)를 참고하세요

:::

## 프로젝트 생성하기

<OrganizationAdminPermission prefix="프로젝트 생성은" />

1. 조직 페이지에서 왼쪽 사이드바에 "프로젝트" 메뉴를 클릭합니다.
2. "새 프로젝트 생성" 버튼을 클릭합니다.
   <img
     src="/ko/img/console/organization/add-project-btn.png"
     style={{ width: 600, display: 'block' }}
   />
3. 프로젝트 이름과 설명을 입력 후 프로젝트 생성을 완료합니다
   <img
     src="/ko/img/console/organization/add-project-modal.png"
     style={{ width: 600, display: 'block' }}
   />

:::info 샘플 테스트 스크립트

프로젝트를 생성하면, 프로젝트 레포지토리에 샘플 테스트 스크립트가 생성됩니다. 샘플 테스트 스크립트는 실제 테스트 프로젝트를 작성할 때 참고할 수 있습니다.

[게임 테스트 자동화 시작하기](/get-started/game-test) 튜토리얼에서 샘플 테스트 스크립트를 사용해 테스트를 실행하는 방법을 확인할 수 있습니다.

:::

## 프로젝트 변경하기

<ProjectAdminPermission prefix="프로젝트 변경은" />

1. 조직 페이지에서 왼쪽 사이드바에 "프로젝트" 메뉴를 클릭합니다.
2. 변경할 프로젝트를 클릭합니다.
3. "설정" 메뉴를 클릭합니다.
4. 프로젝트 이름, 설명을 업데이트하고, "저장" 버튼을 클릭해 변경을 완료합니다.
   <img
     src="/ko/img/console/organization/add-project-btn.png"
     style={{ width: 600, display: 'block' }}
   />

## 프로젝트 삭제하기

<OrganizationAdminPermission prefix="프로젝트 삭제는" />

:::danger 프로젝트 삭제

프로젝트에는 테스트 결과 등의 중요 데이터가 포함되어 있습니다. 프로젝트 삭제 시 데이터를 잃게 되므로 신중하게 진행하세요.

:::

1. 조직 페이지에서 왼쪽 사이드바에 "프로젝트" 메뉴를 클릭합니다.
2. 삭제할 프로젝트를 클릭합니다.
3. "설정" 메뉴를 클릭합니다.
4. "프로젝트 삭제" 버튼을 클릭하고, 확인 버튼을 클릭해 삭제를 완료합니다.
   <img
     src="/ko/img/console/organization/remove-project.png"
     style={{ width: 600, display: 'block' }}
   />
