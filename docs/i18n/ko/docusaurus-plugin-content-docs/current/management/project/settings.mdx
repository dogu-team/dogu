## 프로젝트 ID

<ProjectMemberPermission prefix="프로젝트 ID 조회는" />

프로젝트 ID는 프로젝트 생성 시 자동으로 생성되며, 변경할 수 없습니다.

1. 프로젝트 페이지의 왼쪽 사이드바에서 "설정" 메뉴를 클릭합니다.
2. 조직 ID와 프로젝트 ID를 확인합니다.
   <img
     src="/ko/img/console/project/project-id.png"
     style={{ width: 600, display: 'block' }}
   />

## 프로젝트 변경하기

<ProjectAdminPermission prefix="프로젝트 변경은" />

1. 프로젝트 페이지의 왼쪽 사이드바에서 "설정" 메뉴를 클릭합니다.
2. 변경할 프로젝트를 클릭합니다.
3. "설정" 메뉴를 클릭합니다.
4. 프로젝트 이름, 설명을 업데이트하고, "저장" 버튼을 클릭해 변경을 완료합니다.
   <img
     src="/ko/img/console/organization/add-project-btn.png"
     style={{ width: 600, display: 'block' }}
   />

## 프로젝트 엑세스 토큰 {#access-token}

프로젝트 엑세스 토큰은 [API](/api)에서 사용하거나 [테스트 자동화](/test-automation)에서 사용할 수 있습니다.

<ProjectAdminPermission prefix="프로젝트 엑세스 토큰 조회 및 재발급은" />

:::danger

프로젝트 엑세스 토큰 재발급 시 기존 조직 엑세스 토큰은 더 이상 사용할 수 없습니다.

:::

1. 프로젝트 페이지의 왼쪽 사이드바에서 "설정" 메뉴를 클릭합니다.
2. 프로젝트를 클릭합니다.
3. "설정" 메뉴를 클릭합니다.
4. 기존 프로젝트 엑세스 토큰을 조회하거나 재발급합니다.
   <img
     src="/ko/img/console/project/access-token.png"
     style={{ width: 600, display: 'block' }}
   />

## 프로젝트 템플릿 변경하기

<OrganizationAdminPermission prefix="프로젝트 템플릿 변경은" />

:::danger 프로젝트 템플릿 변경

프로젝트 템플릿마다 UI가 다르거나 기능이 다를 수 있습니다.

:::

1. 프로젝트 페이지의 왼쪽 사이드바에서 "설정" 메뉴를 클릭합니다.
2. 프로젝트를 클릭합니다.
3. "설정" 메뉴를 클릭합니다.
4. "템플릿 변경하기" 버튼을 클릭 후 변경합니다.
   <img
     src="/ko/img/console/project/project-template.png"
     style={{ width: 600, display: 'block' }}
   />

## 프로젝트 삭제하기

<OrganizationAdminPermission prefix="프로젝트 삭제는" />

:::danger 프로젝트 삭제

프로젝트에는 테스트 결과 등의 중요 데이터가 포함되어 있습니다. 프로젝트 삭제 시 데이터를 잃게 되므로 신중하게 진행하세요.

:::

1. 프로젝트 페이지의 왼쪽 사이드바에서 "설정" 메뉴를 클릭합니다.
2. 삭제할 프로젝트를 클릭합니다.
3. "설정" 메뉴를 클릭합니다.
4. "프로젝트 삭제" 버튼을 클릭하고, 확인 버튼을 클릭해 삭제를 완료합니다.
   <img
     src="/ko/img/console/organization/remove-project.png"
     style={{ width: 600, display: 'block' }}
   />
