프로젝트는 모바일 앱, 웹, 게임을 대상으로 테스트 실행과 결과를 제공합니다.  
프로젝트 내에서는 디바이스 제어, 멤버 관리, 테스트 스크립트 작성, 루틴 관리 등의 테스트 집중적인 기능을 제공합니다.

프로젝트에서 제공하거나 관리하는 기능은 각 문서를 참고하세요.

:::tip

**테스트 결과에 대한 정확한 통계를 위해 한 프로젝트에 다른 앱, 웹, 게임에 대한 테스트를 넣지 않는 것을 적극 권장합니다.**

:::

## 프로젝트 생성하기

:::info

프로젝트 생성에 관한 방법은 <a href='/management/organization/project'>조직 프로젝트 생성하기</a> 항목을 참고하세요

:::

## 프로젝트에 참여하기

만약 조직 내에서 `Member` 권한을 갖고 있다면, 조직 내 `Admin` 권한 이상의 관리자 또는 해당 프로젝트의 `Admin` 권한 이상의 멤버로부터 해당 프로젝트 멤버로 추가되어야 합니다.
