## Gitlab Access token 문서

## Gitlab Access token 발급

self-managed 기준으로 아래 경로에서 토큰을 발급 받을 수 있습니다.

### Personal access token

1. 우측 상단 아이콘 클릭
2. `Edit profile` 클릭
3. `Access Tokens` 클릭

  src="/ko/img/console/project/gitlab-personal-access-token.png"
  style={{ width: 800, display: 'block' }}
/>

### Project access token

1. `Project` 클릭
2. `Settings` 클릭
3. `Access Tokens` 클릭

<img
  src="/ko/img/console/project/gitlab-project-access-token.png"
  style={{ width: 800, display: 'block' }}
/>

### Group access token

1. `Group` 클릭
2. `Settings` 클릭
3. `Access Tokens` 클릭
