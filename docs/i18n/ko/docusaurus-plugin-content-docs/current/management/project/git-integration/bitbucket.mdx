## Repository access token

1. Bitbucket 계정으로 로그인합니다.
2. 연동할 레포지토리에 들어간 후 `Repository settings` > `Access tokens` 로 이동합니다.
3. `Create repository access token` 버튼을 클릭합니다.
   <img
     src="/ko/img/console/project/bitbucket-create-token.png"
     style={{ width: 500, display: 'block' }}
   />
4. 토큰 이름을 입력하고 권한을 설정한 후 `Create` 버튼을 클릭합니다.
   :::info
   권한은 `Repositories` > `Read` 권한 이상이어야 합니다.
   :::
   <img
     src="/ko/img/console/project/bitbucket-token-form.png"
     style={{ width: 500, display: 'block' }}
   />
5. 생성된 토큰을 복사합니다.
   <img
     src="/ko/img/console/project/bitbucket-copy-token.png"
     style={{ width: 500, display: 'block' }}
   />
