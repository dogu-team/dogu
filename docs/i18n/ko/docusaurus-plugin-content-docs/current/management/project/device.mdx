프로젝트에 할당된 디바이스는 테스트를 위한 자산입니다.

:::info 디바이스 할당하기

프로젝트에 디바이스를 할당하려면, <a href='../organization/device-farm/device-management#디바이스를-사용할-프로젝트-추가하기'>프로젝트에 디바이스 할당하기</a> 항목을 참고하세요.

:::

## 스튜디오

<ProjectMemberPermission prefix="할당된 디바이스의 스튜디오는" />

:::info

디바이스 스트리밍 및 원격제어는 디바이스의 연결상태가 `Online` 상태인 경우에만 가능합니다.

:::

1. 프로젝트 페이지의 왼쪽 사이드바에서 "디바이스" 메뉴를 클릭합니다.
2. 스트리밍 및 원격제어할 디바이스를 목록에서 찾아 "Studio" 버튼을 클릭합니다.
   <img
     src="/ko/img/console/project/project-device-studio.png"
     style={{ width: 600, display: 'block' }}
   />
3. 스튜디오 페이지에 진입합니다.
   <img
     src="/ko/img/console/project/studio-enter.png"
     style={{ width: 600, display: 'block' }}
   />

:::info 디바이스 원격제어 기능

디바이스 스트리밍 및 원격제어의 기능에 대한 설명은 [디바이스 원격제어](/device-farm/device/streaming-and-remote-control) 문서를 참고해주세요.

:::

## 할당된 디바이스 삭제

<ProjectAdminPermission prefix="할당된 디바이스 삭제는" />

1. 프로젝트 페이지의 왼쪽 사이드바에서 "디바이스" 메뉴를 클릭합니다.
2. 할당을 삭제할 디바이스를 목록에서 찾아 메뉴 버튼(`···`)을 클릭합니다.
3. "프로젝트에서 삭제"버튼을 클릭 후 확인 버튼을 눌러 삭제를 완료합니다.
   <img
     src="/ko/img/console/project/project-device-dealloc-menu.png"
     style={{ width: 600, display: 'block' }}
   />
