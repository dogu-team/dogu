리모트 테스트를 수행하면 리모트 테스트 결과가 표시됩니다.

**앱 테스트를 진행 시 앱을 업로드해야 진행할 수 있습니다. 자세한 내용은 [앱 관리](./app) 문서를 참고하세요.**

:::info 리모트 테스트

리모트 테스트란 루틴이 아닌 로컬 환경에서 테스트를 수행하는 것을 말합니다.  
리모트 테스트는 [튜토리얼](/get-started/tutorials) 문서에서 확인할 수 있습니다.

:::

1. 리모트 테스트를 수행한 후, 프로젝트 페이지의 왼쪽 사이드바에서 "리모트" 메뉴를 클릭합니다.
2. 리모트 테스트 결과를 확인할 수 있습니다.

## 리모트 테스트 결과 상세

<img
  src="/ko/img/console/project/remote-result-detail.png"
  style={{ width: 800, display: 'block' }}
/>

- 왼쪽에서는 리모트 테스트를 수행한 디바이스 목록을 확인할 수 있습니다.
- 디바이스를 선택하면 디바이스의 리모트 테스트 결과를 확인할 수 있습니다.
