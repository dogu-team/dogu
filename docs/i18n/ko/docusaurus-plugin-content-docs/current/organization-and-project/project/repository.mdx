프로젝트는 테스크 스크립트 파일들을 관리하는 레포지토리를 제공합니다. Git을 이용하여 버전, 파일들을 관리합니다.

레포지토리에 관한 내용은 [테스트 스크립트](/script-and-routine/test-script/about)문서를 참고하세요.

1. 프로젝트 페이지에서 "레포지토리" 탭을 클릭합니다.
