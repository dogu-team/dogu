:::info PREREQUISITES

- 프로젝트에 할당된 디바이스 또는 퍼블릭 디바이스가 1대 이상 있어야 합니다. [프로젝트에 디바이스 추가하기](/organization-and-project/organization/device/device-management#디바이스를-사용할-프로젝트-추가하기) 문서를 참고하세요.
- 앱을 프로젝트에 업로드하거나, 디바이스에 설치되어있어야 합니다. 앱 업로드는 [앱 관리](./app) 문서를 참고하세요.

:::

Dogu 콘솔에서 루틴을 만들지 않더라도, 작성한 테스트 스크립트를 실행해 볼 수 있습니다.

:::note

스크립트 테스트에서는 스크립트 편집을 지원하지 않습니다. 스크립트 편집은 [테스트 스크립트](/script-and-routine/test-script/about) 문서를 참고하세요.

:::

## 테스트 실행하기

:::danger 작업중인 디바이스

스크립트 테스트 또한 디바이스 작업에 해당됩니다. 디바이스가 루틴 작업이나 다른 스크립트 테스트를 실행하고 있을 경우, 스크립트 테스트는 수행할 수 없습니다.

디바이스가 루틴을 실행중인지 꼭 확인하고 실행해 주세요.

:::

1. 프로젝트 내에서 "스크립트 테스트" 탭을 클릭합니다.
   <img
     src="/ko/img/console/project/script-test-tab.png"
     style={{ width: 600, display: 'block' }}
   />
2. 왼쪽 파일 탐색기에서 테스트할 스크립트를 선택합니다.

   <img
     src="/ko/img/console/project/select-test-script.png"
     style={{ width: 800, display: 'block' }}
   />

   <br />

   :::info 파일 탐색기

   파일 탐색기에 보이는 폴더는 레포지토리 설정에서 지정한 폴더입니다. 레포지토리 설정 문서를 참고하세요.

   :::

3. 오른쪽 상단에서 테스트 스크립트를 수행할 디바이스를 선택합니다.

   <img
     src="/ko/img/console/project/select-device.png"
     style={{ width: 800, display: 'block' }}
   />

   <br />

   :::info 디바이스 목록

   선택 상자에 보이는 디바이스 목록은 프로젝트에 할당된 디바이스 또는 퍼블릭 디바이스 중 연결상태가 `Online`인 디바이스입니다.

   :::

4. _(선택)_ 앱 목록에서 실행할 앱을 선택합니다.

   <img
     src="/ko/img/console/project/script-select-app.png"
     style={{ width: 800, display: 'block' }}
   />

   <br />

   :::tip 앱 미지정 시 동작

   앱을 선택하지 않으면, 현재 디바이스에서 실행중인 앱을 사용합니다. 사용하는 앱은 [Gamium Engine SDK](https://gamium.dogutech.io/docs/engine/unity/project-configuration)가 설치되어 있어야 합니다.

   **테스트 스크립트가 실행될 수 있도록 앱을 실행해 주세요.**

   :::

   :::info 앱 업로드

   업로드 된 앱이 없다면, [앱 관리](./app) 문서를 참고하여 앱을 업로드하세요.

   :::

5. 활성화된 "실행" 버튼을 클릭합니다.
   <img
     src="/ko/img/console/project/script-run.png"
     style={{ width: 800, display: 'block' }}
   />

:::tip 실행 중지

테스트 스크립트가 실행 중일 때, "실행" 버튼은 "중지" 버튼으로 변경됩니다. "중지" 버튼을 클릭하면 테스트 스크립트가 중지됩니다. 완전히 테스트가 종료되면, 다시 "실행" 버튼으로 바뀌게 됩니다.

<img
  src="/ko/img/console/project/script-stop.png"
  style={{ width: 700, display: 'block' }}
/>

:::

## 메뉴

### 게임 UI 인스펙터

스크립트 테스트에서는 Gamium Engine SDK가 설치된 앱의 게임 오브젝트를 조회할 수 있습니다. 게임 UI 인스펙터에 대한 기능은 [게임 UI 인스펙터](/host-and-device/device/streaming-and-remote-control/game-ui-inspector) 문서를 참고하세요.

<img
  src="/ko/img/console/project/script-ui-inspector.png"
  style={{ width: 700, display: 'block' }}
/>

### 테스트 스크립트 로그

스크립트 테스트가 수행중일 때, 실시간 스크립트 로그를 출력합니다. 이를 통해 스크립트 수행의 성공 여부, 실패 위치 등을 알 수 있습니다.

<img
  src="/ko/img/console/project/script-log.png"
  style={{ width: 700, display: 'block' }}
/>
