조직은 여러 프로젝트를 관리하고, 호스트와 디바이스를 프로젝트에 할당해서 사용하거나, 조직 멤버를 팀으로 그룹 짓거나 권한을 할당하는 등 조직 리소스를 효율적으로 관리할 수 있는 공간입니다.

한 계정에 여러 조직을 생성하거나, 초대를 받아 가입할 수 있습니다.

조직에서 제공하거나 관리하는 기능은 각 문서를 참고하세요.

## 조직 생성하기

1. 콘솔에 로그인 또는 회원가입을 완료합니다.
   - 이메일 인증을 받지 않은 경우, 이메일 인증을 먼저 완료해 주세요.
2. "내 조직" 페이지로 이동합니다.
   - 이미 기존 조직 페이지에 있는 경우, 상단 헤더 메뉴의 프로필 이미지 -> "모든 조직" 메뉴를 클릭합니다.
     <img
       src="/ko/img/console/organization/move-to-my-orgs.png"
       style={{ display: 'block', width: 600 }}
     />
3. "새 조직" 버튼을 클릭합니다.
4. 조직 이름을 입력하고 생성을 완료합니다.
   <img
     src="/ko/img/console/organization/creat-org.png"
     style={{ display: 'block', width: 600 }}
   />

## 조직에 초대받기

조직에 초대받기 위해서는 수신이 가능한 이메일로 초대장을 받아야 합니다. 초대장을 받으려면 해당 조직의 Admin 권한 이상의 관리자에게 문의하세요.

1. 초대장을 받아, 이메일 내 수락 버튼을 클릭합니다.
2. 로그인 또는 회원가입을 완료합니다.
