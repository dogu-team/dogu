# Dost 시작하기

이 페이지에서는 [`호스트`](../introduction#host-and-dost)에 설치되는 프로그램인 [`Dost`](../introduction#host-and-dost)를 시작하는 방법에 대해 다루고 있습니다.

### 요구사항

### 설치

[Dost 다운로드](https://dogutech.io/downloads/dost) 페이지에서 `Dost`를 다운로드 받을 수 있습니다.  
원하는 Platform, Architecture에 맞게 다운로드 받아주세요.

<BrowserOnly>
  {() => (
    <Tabs groupId="os" defaultValue={getBrowserPlatform('windows')}>
      <TabItem value="windows" label="Windows" default>
        <ul>
          <li>
            exe 설치파일을 실행하고, 추가 네트워크 액세스 허용을 동의해주세요.
          </li>
        </ul>

  )}
</BrowserOnly>

### 실행

1. 최초 실행 시 이용약관 및 Third Party License를 확인하고 동의해야 합니다.
2. Dost는 필요한 툴들이 설치되었는지 확인하고 설치되지 않았다면 설치창을 표시합니다. 설치되지 않은 툴들이 있다면 설치를 진행해 주세요.
3. 원활한 Dost 사용을 위해서는 설치하고자 하는 운영체제에 맞는 권한이 필요합니다. 권한이 허용되지 않았다면 권한을 허용해 주세요.

### Console 연결

[호스트 관리](/organization-and-project/organization/host) 페이지를 확인하여 호스트 토큰을 획득하고, 획득한 토큰을 Dost에 입력해주세요.

### 설정

연결이 되었다면 Dost는 설정창으로 이동합니다. 설정창에서는 Dost의 기본 설정을 변경할 수 있습니다.

export const GeneralBlock = ({ platform }) => (
  <div>
    <h4>General</h4>
    <br />

      <img src={dost_windows_setting} style={{ width: 600 }} />

      <img src={dost_macos_setting} style={{ width: 600 }} />

    <ul>
      <li>
        Start Dost when you log in
        <ul>
          <li>PC에 로그인시 Dost가 자동으로 시작합니다.</li>
        </ul>
      </li>
    </ul>
  </div>
);
