## 디바이스 연결 확인하기

### 케이블 확인하기

1. 데이터 전송 가능한 케이블인지 확인해주세요
2. 케이블을 뽑고 재연결합니다.

### 디바이스 설정 확인하기

1. [디바이스 설정](/host-and-device/device/settings) 문서를 따라 디바이스를 설정했는지 확인해주세요.

## 호스트 확인하기

Dost를 아래 방법을 사용해 확인해주세요.

### Dost 재실행하기

Dost를 완전히 종료 후, 다시 실행합니다. 디바이스가 다시 연결되는지 확인해보세요.

:::caution

Dost를 재실행하면 해당 호스트와 연결된 디바이스들의 연결상태가 `Offline`이 됩니다.

:::

### 최신 Dost인지 확인하기

Dost가 최신 버전인지 확인해주세요. Dost 최신 버전은 [여기](https://dogutech.io/downloads/dost)에서 다운로드 받을 수 있습니다.

### 호스트 토큰 재발급받기

[호스트 관리](/organization-and-project/organization/host)문서를 통해 호스트 토큰을 재발급받아 호스트를 다시 등록해보세요.
