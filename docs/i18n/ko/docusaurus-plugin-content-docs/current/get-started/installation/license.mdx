## Self-Hosted 라이센스

Self-Hosted 환경에서는 Community, Professional 라이센스를 사용할 수 있습니다.  
라이센스 별 기능, 가격에 대한 내용은 [Dogu Pricing](https://dogutech.io/pricing?type=self-hosted) 페이지를 참고해주세요.

**기본 라이센스는 Community 라이센스이며, Professional 라이센스는 별도의 라이센스를 구매해야 사용할 수 있습니다.**  
[Professional 라이센스](https://dogutech.io/pricing?type=self-hosted) 페이지에서 체험 신청 및 문의를 통해 Professional 라이센스를 구매할 수 있습니다.

### 라이센스 등록 전 준비사항

라이센스 인증을 위해 `https://api-client-admin.dogutech.io`의 네트워크 접근을 허용해주세요.

:::info 접근 확인 방법
[Request]

```
curl -v -X GET "https://api-client-admin.dogutech.io"
```

### 라이센스 등록 방법

1. 루트 계정으로 로그인합니다.
2. 상단 헤더의 프로필 메뉴에서 "관리자 설정" 메뉴를 클릭합니다.
   <img
     src="/ko/img/get-started/admin-setting-menu.png"
     style={{ width: 500, display: 'block' }}
   />
3. 발급받은 라이센스 키를 입력하고 "라이센스 등록" 버튼을 클릭합니다.
   <img
     src="/ko/img/get-started/license-activate-form.png"
     style={{ width: 600, display: 'block' }}
   />
4. 정상적으로 라이센스가 등록되었는지 확인합니다.
   <img
     src="/ko/img/get-started/license-activate-confirm.png"
     style={{ width: 600, display: 'block' }}
   />

:::info

등록 과정에 문제가 발생하면 [문의하기](https://dogutech.io/contact-us)를 통해 문의해주세요.

:::
