:::info PREREQUISITES
디바이스를 사용하기 전에 호스트가 추가되어 있어야 합니다. 호스트를 추가하는 방법은 [호스트 추가하기](/get-started/tutorials/device-farm/host) 문서를 참고하세요.
:::

디바이스를 추가하고 태그로 묶어 디바이스 팜을 구축합니다. 추가된 디바이스는 테스트 수행 및 원격제어 등의 작업을 수행할 수 있습니다.

### 디바이스 사용하기

1. Dogu Agent가 설치된 Host 장비에 [Android 디바이스 연결](/device-farm/device/settings?platform=android)을 합니다.
2. 홈페이지에서 조직 내 왼쪽 사이드바에서 디바이스 메뉴를 클릭하고 "대기중인 디바이스" 탭을 클릭합니다.
3. 정상적으로 연결이 된 경우 사용할 수 있는 디바이스가 보이며, 사용할 디바이스의 메뉴 버튼(`···`)을 클릭 후 "사용하기" 메뉴를 클릭하세요.
   <img
     src="/ko/img/get-started/device-start-using.png"
     style={{ width: 600, display: 'block' }}
   />
4. 추가할 디바이스를 사용할 프로젝트를 선택 후, 추가를 완료해줍니다.
   <img
     src="/ko/img/get-started/add-project-modal.png"
     style={{ width: 600, display: 'block' }}
   />
5. "사용중인 디바이스" 탭을 클릭해 디바이스가 추가되었는지 확인합니다.
   <img
     src="/ko/img/get-started/check-use-device.png"
     style={{ width: 600, display: 'block' }}
   />

:::caution 권한 및 퍼블릭 디바이스

Dogu 콘솔에는 조직 내 권한 기능이 있습니다. 권한 기능을 이용해 테스트 결과를 보거나, 디바이스에 접근할 수 있는 멤버를 자유롭게 지정하세요.

퍼블릭 디바이스의 경우에는 모든 멤버 또는 모든 프로젝트에서 해당 디바이스에 접근(스트리밍 및 원격제어, 테스트 실행 등)이 가능합니다.

권한 기능에 대해서는 [권한 시스템](/management/organization/permission) 문서를 참고하세요

:::

:::info 더 알아보기

디바이스 추가와 관련한 더 자세한 설명은 [조직 디바이스 관리](/management/organization/device-farm/device-management#디바이스-추가하기) 문서를 참고하세요.

:::

### 디바이스 태그 붙이기

디바이스 태그를 통해 디바이스를 그룹화하고 테스트를 실행할 수 있습니다.

:::tip 기본 태그

디바이스를 사용할 때 기본적으로 디바이스의 OS가 태그로 붙습니다.

:::

1. 콘솔에서 조직 내 왼쪽 사이드바에서 디바이스 메뉴를 클릭합니다.
2. 태그를 붙일 디바이스의 메뉴 버튼(`···`)을 클릭하고, "태그 변경" 메뉴를 클릭합니다.
   <img
     src="/ko/img/get-started/edit-tag-menu.png"
     style={{ width: 600, display: 'block' }}
   />
3. 생성할 태그 이름을 입력하고, 태그 만들기 버튼을 클릭합니다.
4. 모달을 닫고, 디바이스에 태그가 추가되었는지 확인합니다.
   <img
     src="/ko/img/get-started/tag-check.png"
     style={{ width: 400, display: 'block' }}
   />

:::info 더 알아보기

디바이스 태그 관련 더 자세한 정보는 [조직 디바이스 태그 관리](/management/organization/device-farm/tag-management) 문서를 참고하세요.

:::

### 디바이스 스트리밍하기

추가된 디바이스는 스트리밍 또는 원격 제어 기능을 사용할 수 있습니다.

1. 콘솔에서 조직 내 왼쪽 사이드바에서 디바이스 메뉴를 클릭합니다.
2. 스트리밍할 디바이스의 메뉴 버튼(`···`)을 클릭하고, "스트리밍" 메뉴를 클릭합니다.
   <img
     src="/ko/img/get-started/streaming-menu.png"
     style={{ width: 600, display: 'block' }}
   />
3. 디바이스 스트리밍이 시작되었습니다. 화면을 클릭하거나, 드래그하여 디바이스를 원격 제어할 수 있습니다.
   <img
     src="/ko/img/get-started/streaming-android.png"
     style={{ width: 600, display: 'block' }}
   />

## 마치며

디바이스 화면 실시간 스트리밍 및 원격제어, 디바이스 태그, 디바이스 프로젝트 할당 기능 등을 이용해 조직 내에서 자유롭게 디바이스를 관리해 보세요.

- 디바이스 스트리밍 및 원격 제어는 [디바이스 원격 제어](/device-farm/device/streaming-and-remote-control) 문서를 참고하세요.
- 디바이스 관리 기능에 대해서는 [디바이스 관리](/management/organization/device-farm/device-management) 문서를 참고하세요.
- 디바이스 태그 관리 기능에 대해서는 [디바이스 태그 관리](/management/organization/device-farm/tag-management) 문서를 참고하세요.
