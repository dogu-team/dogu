이 튜토리얼에서는 테스트가 실행될 디바이스를 디바이스 팜에 연결하는 방법을 설명합니다.

## 조직과 프로젝트 생성하기

디바이스 팜을 사용하기 위해서는 먼저 조직과 프로젝트를 생성해야 합니다.

### 조직 생성하기

1. "내 조직" 페이지로 이동합니다.
   - 이미 기존 조직 페이지에 있는 경우, 상단 헤더 메뉴의 프로필 이미지 -> "모든 조직" 메뉴를 클릭합니다.
     <img
       src="/ko/img/console/organization/move-to-my-orgs.png"
       style={{ width: 400, display: 'block' }}
     />
2. "새 조직" 버튼을 클릭합니다.
3. 조직 이름을 입력하고 생성을 완료합니다.

### 프로젝트 생성하기

1. 조직 페이지에서 왼쪽 사이드바에 "프로젝트" 메뉴를 클릭합니다.
2. "새 프로젝트 생성" 버튼을 클릭합니다.
3. 프로젝트 이름과 설명을 입력 후 프로젝트 생성을 완료합니다
