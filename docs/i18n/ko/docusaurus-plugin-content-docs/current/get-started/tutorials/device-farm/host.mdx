### 호스트 요구사항

| OS      | OS Version                              | Architecture |
| ------- | --------------------------------------- | ------------ |
| Windows | Windows 10, Windows 11                  | x64          |
| macOS   | macOS 11 (Big Sur) or later, Xcode 12.5 | x64, arm64   |

:::caution

요구사항을 만족하지 않는 경우 Dogu Agent 이용에 문제가 있을 수 있습니다.

:::

- `DOGU_HOST:4000` 값을 API URL에 입력합니다.
  <img
    src="/ko/img/get-started/dost-api-url-input.png"
    style={{ width: 600, display: 'block' }}
  />

</TabItem>
</Tabs>

2. 콘솔에서 조직 내 왼쪽 사이드바에서 호스트 메뉴를 클릭합니다.
   <img
     src="/ko/img/get-started/add-host.png"
     style={{ width: 600, display: 'block' }}
   />
3. 호스트 추가 버튼을 클릭하고 호스트 이름을 입력 후 생성을 완료합니다.
4. 발급받은 호스트 토큰을 복사 후, Dogu Agent의 토큰 입력창에 복사한 토큰을 붙여 넣어 연결을 완료합니다.
   <img
     src="/ko/img/get-started/dost-token-input.png"
     style={{ width: 600, display: 'block' }}
   />
5. 콘솔에서 생성한 호스트의 상태가 "Connected" 상태가 되면 호스트 등록이 완료되었습니다.
   <img
     src="/ko/img/get-started/host-connected-check.png"
     style={{ width: 600, display: 'block' }}
   />
