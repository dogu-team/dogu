[Gamium](https://gamium.dogutech.io/)은 게임 테스트 자동화를 위한 라이브러리입니다. Dogu DeviceFarm에 있는 디바이스를 대상으로 Gamium 테스트 스크립트를 실행할 수 있습니다.
이 문서에서는 미리 작성된 Gamium 테스트스크립트를 샘플로 제공하여 설명합니다.

:::info

gamium에 더 알고 싶다면 [Introduction](https://gamium.dogutech.io/docs/get-started/introduction) 문서를 참고해주세요.

:::
