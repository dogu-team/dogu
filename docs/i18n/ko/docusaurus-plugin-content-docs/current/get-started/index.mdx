Dogu는 웹, 모바일, 게임 애플리케이션을 다양한 플랫폼에서 테스트 자동화할 수 있도록 돕는 통합 테스트 자동화 플랫폼입니다.  
여러 인프라 설치 과정 없이 테스트 자동화를 위한 환경을 제공합니다.  

## 기능
Dogu는 테스트 자동화에 필요한 인프라를 설치하고 관리하는 과정을 자동화하여 테스트 자동화에 집중할 수 있도록 돕습니다.
