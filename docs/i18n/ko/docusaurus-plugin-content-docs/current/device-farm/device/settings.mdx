## Android

### 요구사항

- Android 8.0 이상

### 설정

- ↔ 안드로이드 기기와 PC를 USB 케이블로 연결하고, 연결된 상태를 유지해주세요.
  - 데이터 전송을 지원하는 USB 케이블을 사용해주세요.
- 🔍 [개발자 옵션](https://developer.android.com/studio/debug/dev-options)을 활성화해주세요.
- 🐛 [USB 디버깅](https://developer.android.com/studio/run/device#setting-up)을 활성화해주세요.
  - 화면에 `USB 디버깅을 허용하시겠습니까?` 라는 메시지가 뜨면 `이 컴퓨터에서 항상 허용`을 체크하고 `확인`을 눌러 동의해주세요.
- 😌 Google Assistant 등 음성 인식 기능을 비활성화 해주세요.
- 👓 [Google Play Protect](https://support.google.com/googleplay/answer/2812853?hl=en)를 Off해주세요.
- 🔓 비밀번호를 비활성화 해주세요.
- 📱 화면 자동 잠금을 해제해주세요.
- 🔅 밝기를 낮추면 에너지 절약에 도움이 됩니다.

:::note
기기를 추가로 연결할 때마다 USB를 통해 전송되는 데이터의 사이즈가 늘어납니다.  
여러개의 디바이스를 하나의 호스트에 연결할때에는 되도록 유전원 USB 허브를 사용해주세요.

:::

</TabItem>
<TabItem value="android-emulator" label="Android(Emulator)">

## Android(Emulator)

### 요구사항

- Android Studio 2022.1.1 이상

### 설정

- [Create and manage virtual devices](https://developer.android.com/studio/run/managing-avds) 문서를 참고하여 가상 디바이스를 생성해주세요.
- Select a system image 단계에서 API Level 26 이상으로 설정해주세요.
- Create a hardware profile 단계에서 RAM을 4GB 이상으로 설정해주세요.

</TabItem>

## iOS

### 요구사항

- iOS 14.0 이상

### 설정

- ↔ iOS 기기와 PC를 USB 케이블로 연결하고, 연결된 상태를 유지해주세요.
  - 이후 기기 화면에 [이 컴퓨터를 신뢰하겠습니까?](https://support.apple.com/ko-kr/HT202778) 라는 메시지가 뜨면 `신뢰`를 눌러 동의해주세요.
- 🔓 비밀번호를 비활성화 해주세요.
- 📱 화면 자동 잠금을 해제해주세요.
- 🔅 밝기를 낮추면 에너지 절약에 도움이 됩니다.
- 🌐 설정 -> Safari -> 고급에서 웹 속성을 활성화해주세요.
- 🧑‍💻 iOS 16 이상인 경우에 [Developer mode](https://developer.apple.com/documentation/xcode/enabling-developer-mode-on-a-device)를 enable 해주세요.

:::note
기기를 추가로 연결할 때마다 USB를 통해 전송되는 데이터의 사이즈가 늘어납니다.  
여러개의 디바이스를 하나의 호스트에 연결할때에는 되도록 유전원 USB 허브를 사용해주세요.

:::

</TabItem>
