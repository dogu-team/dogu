Dogu 콘솔에서 디바이스 로그를 확인할 수 있습니다.

<img
  src="/ko/img/console/device-streaming/device-logs.png"
  style={{ width: 600, display: 'block' }}
/>

## 사용 방법

1. Dogu 콘솔에서 디바이스 스트리밍 페이지를 접속합니다.
2. "디바이스 로그" 탭을 선택합니다.
3. 디바이스 로그를 확인합니다.
4. 필요에 따라 필터링 할 문자열을 설정합니다. 필터 결과는 저장 버튼 클릭 이후에 반영됩니다.

- 일시정지 버튼: 디바이스 로그를 일시정지합니다.
- 삭제 버튼: 디바이스 로그를 삭제합니다.
