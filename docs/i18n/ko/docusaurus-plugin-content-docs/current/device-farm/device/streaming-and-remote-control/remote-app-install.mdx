Dogu 콘솔에서 디바이스에 원격으로 앱 설치를 할 수 있습니다.

:::info 지원 확장자

`.apk` 확장자를 지원합니다.

:::

## 사용 방법

1. Dogu 콘솔에서 디바이스 스트리밍 페이지에 접속합니다.
2. "원격 앱 설치" 메뉴를 클릭합니다.
3. 파일을 선택하거나 드래그 앤 드롭으로 파일을 업로드합니다.
4. "앱 업로드 -> 앱 조각 모음 -> 앱 설치" 단계를 거쳐 앱이 설치됩니다.
   <img
     src="/ko/img/console/device-streaming/app-installation.png"
     style={{ width: 600, display: 'block' }}
   />

:::tip

설치 후 앱 열기 옵션을 통해 앱을 바로 실행할 수 있습니다.

:::
