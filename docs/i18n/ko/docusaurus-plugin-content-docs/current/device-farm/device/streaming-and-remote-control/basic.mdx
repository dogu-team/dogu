## 디바이스 화면 터치 입력

Dogu 콘솔에서 비디오 영역을 터치하면 디바이스 화면에 터치 이벤트가 전달됩니다.

## 키보드 입력

Dogu 콘솔에서 키보드 입력을 하면 디바이스 화면에 키보드 이벤트가 전달됩니다.

영문과 숫자, 특수문자 등을 지원합니다.

:::tip

<Tabs groupId="platform" defaultValue="android" queryString>
<TabItem value="android" label="Android">

- 한글 등의 지역 언어를 입력하려면 <img src="/ko/img/device/gboard.png" style={{ width: 30, position:'relative', bottom:'-5px'}} /> [`Gboard`](https://play.google.com/store/apps/details?id=com.google.android.inputmethod.latin)를

- `CMD(WIN)` + `/` 키를 눌러 키보드 단축키들을 확인할 수 있습니다.

</TabItem>
<TabItem value="ios" label="iOS">

- 만약 App Switcher와 기능을 사용하고 싶다면 [AssistiveTouch](https://support.apple.com/en-us/HT202658)를 활용해주세요.

## 디바이스 버튼 입력

Dogu 콘솔에서 디바이스 화면 오른쪽의 디바이스 버튼을 클릭하면 디바이스 화면에 버튼 이벤트가 전달됩니다.

디바이스 버튼은 다음과 같습니다.

- 볼륨 업/다운
- 음소거
- 화면 잠금
- 화면 잠금 해제
- 전원
- 스크린 캡처
- 뒤로가기 (Android)
- 홈 (Android)
- 앱 스위치 (Android)
