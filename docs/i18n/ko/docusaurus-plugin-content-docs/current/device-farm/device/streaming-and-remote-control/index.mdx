Dogu는 사용자가 디바이스와 물리적 공간이 달라도 웹 페이지에서 디바이스를 관리하고 원격으로 제어할 수 있는 기능을 제공합니다.

디바이스 스트리밍 및 원격제어에서 제공하는 기능은 다음과 같습니다.

1. 기본 기능
   - 디바이스 화면 스트리밍
   - 디바이스 터치 입력
   - 키보드 입력
   - 디바이스 버튼 입력(소리, 전원, 앱 스위칭, 홈 등)
2. 디바이스 프로파일 정보(CPU, MEM)
3. UI 인스펙터
4. 원격 앱 설치
5. 디바이스 로그캣

:::tip

위 기능들을 이용하기 위해 디바이스는 **반드시** `Online` 연결상태를 유지해야 하고, 한 개 이상의 프로젝트에 할당을 하거나, 퍼블릭 디바이스이어야 합니다.

디바이스를 사용하는 방법은 [조직 디바이스 관리](/management/organization/device-farm/device-management) 또는 [호스트 관리](/management/organization/device-farm/host-management) 문서를 참고하세요.

:::

## 디바이스 스튜디오

:::info 스튜디오 사용 권한

디바이스 스튜디오 기능은 조직 내 권한 및 프로젝트의 멤버 여부에 따라 이용할 수 있습니다.

[조직 권한 시스템](/management/organization/permission) 및 [프로젝트 권한 시스템](/management/project/permission) 문서를 참고하세요.

:::

1. 프로젝트 페이지 내 왼쪽 사이드바에서 "디바이스" 메뉴를 클릭합니다.
   - 또는 "Studio" 메뉴를 클릭해 스튜디오에 진입합니다.
2. 연결상태가 `Online`인 디바이스 중에서 "Studio" 버튼을 클릭합니다.
