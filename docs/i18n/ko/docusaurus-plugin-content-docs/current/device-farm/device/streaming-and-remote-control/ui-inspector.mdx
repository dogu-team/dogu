## Native UI Inspector

Dogu 디바이스 팜에 등록된 디바이스라면 UI 인스펙터를 사용할 수 있습니다.

    width="600"

### 사용방법

1. Dogu 콘솔에서 디바이스 스트리밍 페이지에 접속 후, "인스펙터" 메뉴를 클릭합니다.
2. 컨텍스트 중 `NATIVE_APP`을 선택합니다.
3. UI 트리를 펼처 UI 요소를 보거나, <svg viewBox="64 64 896 896" focusable="false" data-icon="select" width="1em" height="1em" fill="currentColor" aria-hidden="true"><path d="M880 112H144c-17.7 0-32 14.3-32 32v736c0 17.7 14.3 32 32 32h360c4.4 0 8-3.6 8-8v-56c0-4.4-3.6-8-8-8H184V184h656v320c0 4.4 3.6 8 8 8h56c4.4 0 8-3.6 8-8V144c0-17.7-14.3-32-32-32zM653.3 599.4l52.2-52.2a8.01 8.01 0 00-4.7-13.6l-179.4-21c-5.1-.6-9.5 3.7-8.9 8.9l21 179.4c.8 6.6 8.9 9.4 13.6 4.7l52.4-52.4 256.2 256.2c3.1 3.1 8.2 3.1 11.3 0l42.4-42.4c3.1-3.1 3.1-8.2 0-11.3L653.3 599.4z"></path></svg> 버튼을 누른 후, 디바이스 스트리밍 화면에 마우스를 올리고, 클릭하면 상세정보를 볼 수 있습니다.

## Game UI Inspector

Gamium Engine SDK가 빌드에 포함된 개임의 경우, Dogu 콘솔에서 UI 인스펙터를 사용해 UI를 볼 수 있습니다.

:::info Gamium Engine

게임 빌드에 Gamium Engine SDK를 포함시키는 방법은 [Gamium Engine](https://gamium.dogutech.io/docs/engine/unity/project-configuration) 문서를 참고하세요.

:::

<div align="center">
  <img
    src="https://github.com/dogu-team/dogu/blob/main/.github/resources/gamium-inspecting.gif?raw=true"
    width="600"
  />
</div>

### 사용방법

1. 디바이스에서 게임 앱을 실행합니다.
2. Dogu 콘솔에서 디바이스 스트리밍 페이지에 접속 후, "인스펙터" 메뉴를 클릭합니다.
3. 컨텍스트 중 `GAMIUM`을 선택합니다.
   :::tip
   컨텍스트에서 `GAMIUM`이 보이지 않는다면 `Reconnect` 버튼을 눌러 다시 연결해보세요.
   :::
4. UI 트리를 펼처 게임 오브젝트를 보거나, <svg viewBox="64 64 896 896" focusable="false" data-icon="select" width="1em" height="1em" fill="currentColor" aria-hidden="true"><path d="M880 112H144c-17.7 0-32 14.3-32 32v736c0 17.7 14.3 32 32 32h360c4.4 0 8-3.6 8-8v-56c0-4.4-3.6-8-8-8H184V184h656v320c0 4.4 3.6 8 8 8h56c4.4 0 8-3.6 8-8V144c0-17.7-14.3-32-32-32zM653.3 599.4l52.2-52.2a8.01 8.01 0 00-4.7-13.6l-179.4-21c-5.1-.6-9.5 3.7-8.9 8.9l21 179.4c.8 6.6 8.9 9.4 13.6 4.7l52.4-52.4 256.2 256.2c3.1 3.1 8.2 3.1 11.3 0l42.4-42.4c3.1-3.1 3.1-8.2 0-11.3L653.3 599.4z"></path></svg> 버튼을 누른 후, 디바이스 스트리밍 화면에 마우스를 올리고, 클릭하면 상세정보를 볼 수 있습니다.

### 오브젝트 정보

UI 트리에서 게임 오브젝트를 클릭하거나, <svg viewBox="64 64 896 896" focusable="false" data-icon="select" width="1em" height="1em" fill="currentColor" aria-hidden="true"><path d="M880 112H144c-17.7 0-32 14.3-32 32v736c0 17.7 14.3 32 32 32h360c4.4 0 8-3.6 8-8v-56c0-4.4-3.6-8-8-8H184V184h656v320c0 4.4 3.6 8 8 8h56c4.4 0 8-3.6 8-8V144c0-17.7-14.3-32-32-32zM653.3 599.4l52.2-52.2a8.01 8.01 0 00-4.7-13.6l-179.4-21c-5.1-.6-9.5 3.7-8.9 8.9l21 179.4c.8 6.6 8.9 9.4 13.6 4.7l52.4-52.4 256.2 256.2c3.1 3.1 8.2 3.1 11.3 0l42.4-42.4c3.1-3.1 3.1-8.2 0-11.3L653.3 599.4z"></path></svg> 버튼을 누른 후, 디바이스 스트리밍 화면에 마우스를 올리고, 클릭하면 상세정보를 볼 수 있습니다.

<img
  src="/ko/img/console/device-streaming/game-object-detail.png"
  style={{ width: 600, display: 'block' }}
/>

<br />

- Hit Point: 마우스 위치에서 RayCast를 했을 때 충돌되는 월드 좌표입니다.
- Path: 오브젝트의 고유 path를 표시합니다.
- Name: 오브젝트의 이름을 표시합니다.
- Text: 오브젝트 내 텍스트를 표시합니다.
- Screen position: 오브젝트 중심 좌표입니다. (스크린 기준)
- Screen rect size: 오브젝트의 x, y 크기를 표시합니다. (스크린 기준)
- Position: 오브젝트의 월드 좌표를 표시합니다.
- Rotation: 오브젝트의 월드 기준 회전을 표시합니다.
