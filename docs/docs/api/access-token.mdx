This document lists token used in Dogu.

:::danger

Please be cautious not to expose the token to the public.

:::

## Organization Access Token

An token is a key that allows external access to the Dogu API.  
It is automatically generated when creating an organization in Dogu and can be found in the organization's settings page.  
Organization Access Token has Organization Administrator Permission.

:::info

For more information on token management, please refer to the [Organization Settings](/management/organization/settings) documentation.

:::

## Project Access Token

An token is a key that allows external access to the Dogu API. It is automatically generated when creating an organization in Dogu and can be found in the organization's settings page.
