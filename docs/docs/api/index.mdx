You can use the Dogu API externally using an API token.

## API Token

:::danger

Please be cautious not to expose the API token to the public.

:::

An API token is a key that allows external access to the Dogu API. It is automatically generated when creating an organization in Dogu and can be found in the organization's settings page.

:::info

For more information on token management, please refer to the [Organization Settings](/management/organization/settings) documentation.

:::
