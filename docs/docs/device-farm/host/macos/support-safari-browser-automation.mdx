# Support Safari Browser Automation

## Enable safaridriver

Type the following command in the terminal to enable safaridriver

```shell
sudo /usr/bin/safaridriver --enable
```

:::tip Official documentation

see [Testing with WebDriver in Safari](https://developer.apple.com/documentation/webkit/testing_with_webdriver_in_safari#2957277) for more information.

:::
