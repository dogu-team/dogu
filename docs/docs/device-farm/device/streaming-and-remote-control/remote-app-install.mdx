You can remotely install apps on the device using the Dogu console.

:::info Supported Extension

Supports the `.apk` extension.

:::

## Usage

1. Access the Device Streaming page in the Dogu console.
2. Click on the "Remote App Installation" menu.
3. Select or drag and drop the file to upload.
4. The app will be installed through the "App Upload -> App Bundle -> App Installation" steps.
   <img
     src="/img/console/device-streaming/app-installation.png"
     style={{ width: 600, display: 'block' }}
   />

:::tip

After installation, you can directly launch the app using the "Open app" option.

:::
