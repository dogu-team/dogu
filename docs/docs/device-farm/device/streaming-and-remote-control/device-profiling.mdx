## Profiling information

### Foreground app

The name of the currently running foreground app process.

### CPU

The CPU usage of the device.

:::tip

You can check the CPU usage of the foreground app.

:::

### Memory

The memory usage of the device.

:::tip

You can check the memory usage of the foreground app.

:::
