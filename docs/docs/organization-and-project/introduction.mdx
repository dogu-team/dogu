Introducing the features provided by the Dogu console.

## Organization

An organization is a unit within the Dogu Console that allows you to access and utilize the features provided by Dogu.

Within an organization, you can manage projects, devices, teams, members, and more.

## Project

Projects belong to organizations and allow you to run automated tests.

Within a project, you can manage test scripts, routines, project members, and more.
