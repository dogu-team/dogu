That document explains how to manage your test scripts.

## Dogu Gitlab

Dogu provides its own [Gitlab](https://gitlab.dogutech.io) to manage test scripts.
When you sign up as a member, you automatically join Dogu Gitlab, and the Gitlab service is linked with the Dogu service.

:::info
The Dogu Gitlab ID and password are the same as the Dogu service ID and password.
:::
