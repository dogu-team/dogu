Test scripts are stored in Dogu Gitlab. There are two ways to manage test scripts.

- **Web IDE**
- **Local (Git)**

# Web IDE

The Web IDE allows you to write and edit test scripts on web pages.
The Web IDE is available by clicking the **Edit** button on the **Project** > **Script** page.

# Local (Git)

You can clone the test script repository using [Git](https://git-scm.com/downloads).

1. On the **project** page, click the **Gitlab** button to go to the project's Gitlab page.

2. Click the **Clone with HTTPS** button on the Gitlab page to copy the clone URL.

3. On the [Account](https://dogutech.io/account) page, copy the **token** from the **Gitlab** section.

4. Add `oauth2:{token}@` after `https://` in the clone URL.

5. Receive Clone using the command below in Terminal.

```
git clone <Clone URL>
```

Example.

```
git clone https://oauth2:{token}@gitlab.dogutech.io/{my-organization-id}/{my-project-id}.git
```

:::tip

**In addition to the Git CLI, other Git Clients are also available.**

:::
