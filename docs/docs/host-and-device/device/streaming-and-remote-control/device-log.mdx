## Usage

1. Access the Device Streaming page on the Dogu console.
2. Select the "Device logs" tab.
3. Check the device logs.
4. Set the filter string for filtering if needed. The filter results will be applied after clicking the Save button.

- Pause Button: Pause the device logs.
- Clear Button: Clear the device logs.
