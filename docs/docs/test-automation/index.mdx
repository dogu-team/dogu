The Dogu Agent includes Selenium Grid, Appium Server, and more, allowing tests to be remotely executed on devices present in the device farm through the included servers.

## Selenium Grid

Selenium Grid is a tool that enables remote control of browsers. To learn more about Selenium Grid options, please refer to [Selenium](/test-automation/selenium).

## Appium Server

Appium Server is a tool that enables remote control of applications. To learn more about Appium Server options, please refer to [Appium](/test-automation/appium).

## Playwright

Playwright is a tool that enables remote control of browsers. To learn more about Playwright options, please refer to [Playwright](/test-automation/playwright).
