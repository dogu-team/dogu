## Remote Appium options

Describes options when running Appium. Please refer to the [Tutorial](/get-started/tutorials/test-automation/remote-testing/appium) on how to run Appium remote tests.

## Dogu options

[Extension capabilities](https://www.w3.org/TR/webdriver/#dfn-extension-capability) that extend the existing capabilities of Webdriver to suit Dogu. The keys below are configurable fields in the `dogu:options` namespace.

| key              | required | description                                                                                                                                                                  |
| ---------------- | -------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `token`          | true     | Please refer to the [API token documentation](/management/organization/settings#api-token) to obtain and enter an API token.                                                 |
| `organizationId` | true     | The organization's ID. You can easily find it on the project settings page. See [the project setup documentation](/management/project/settings)                              |
| `projectId`      | true     | ID of the project. You can easily find it on the project settings page. See [the project setup documentation](/management/project/settings)                                  |
| `runsOn`         | true     | Used to set the target to run tests on. Please enter the name or tag of a specific device. If multiple are set, targets that satisfy all conditions are set.                 |
| `appVersion`     | false    | This is the version of the app you want to test among the apps uploaded to Project. Refer to [the app management page](/management/project/app) and upload the app you want. |
| `browserName`    | false    | Used for browser testing. (ex. chrome)                                                                                                                                       |
| `browserVersion` | false    | Used for browser testing. (ex. 91.0.4472.114)                                                                                                                                |

## Overriden Capabilities

Capabilities automatically set by Dogu Server.

| key                        | description                                                                                    | value                                            |
| -------------------------- | ---------------------------------------------------------------------------------------------- | ------------------------------------------------ |
| `platformName`             | The `platformName` is automatically overwritten by the Dogu server through device information. | android: `android`<br></br> ios: `ios`           |
| `browserName`              | If `dogu:options.browserName` exists, it will be overwritten                                   | {browserName}                                    |
| `browserVersion`           | If `dogu:options.browserVersion` exists, it will be overwritten                                | {browserVersion}                                 |
| `appium:automationName`    | Dogu server automatically overwrites with device information                                   | android: `uiautomator2`<br></br> ios: `xcuitest` |
| `appium:deviceName`        | Device serial or udid                                                                          | android: {serial}<br></br> ios: {udid}           |
| `appium:udid`              | Device serial or udid                                                                          | android: {serial}<br></br> ios: {udid}           |
| `appium:mjpegServerPort`   | host machine's free random port                                                                | {free random port}                               |
| `appium:app`               | Downloaded app path from Dogu platform. if `dogu:options.appVersion` is exist.                 | {downloaded app path}                            |
| `appium:systemPort`        | **ANDROID ONLY** host machine's free random port                                               | {free random port}                               |
| `appium:chromedriverPort`  | **ANDROID ONLY** host machine's free random port                                               | {free random port}                               |
| `appium:webDriverAgentUrl` | **IOS ONLY** Host port forwarded to device WebdriverAgent server                               | http://127.0.0.1:{idle random port}              |
| `appium:showXcodeLog`      | **IOS ONLY** Whether to show Xcode log to Dogu Agent                                           | true                                             |
| `appium:derivedDataPath`   | **IOS ONLY** Derived data path                                                                 | {os temp path}                                   |
