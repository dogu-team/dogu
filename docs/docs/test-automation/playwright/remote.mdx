## Remote Playwright (Experimental 🧪)

:::note

This is an experimental 🧪 feature.  
Currently, you must put the value of the environment variable yourself... 😿  
We will improve this in the future to make it easier 😆 to use.
And we will support other browser vendors! 🦊🐧🦁  
(If you want to use other browser vendors, please contact us!)

:::
