## Prerequisites

- You need to have Dogu Access key, which you can find in your organization profile.
- Node.js 16 or higher, npm must be installed on your computer.

## Run a sample script

### Clone sample repository

```sh
git clone https://github.com/dogu-team/dogu-examples
cd dogu-examples/app/mobile/appium/javascript
```

### Install the dependencies

```sh
npm install
```

### Upload sample app to dogu

- In order to run the test script, the test target app must be uploaded to dogu.
- The app to be uploaded to dogu can be downloaded from the following link. [link](https://github.com/dogu-team/third-party-binaries/releases/download/wikipedia-apps/wikipedia-android.apk)
  - Refering to the [App Management](/management/project/app) page upload the app.

### Configure dogu options

- Fill in the values below in the test script.
  - `accessKey`:
    - Please refer to [API token](/management/organization/settings#api-token) to obtain and enter an API token..
  - `organizationId`, `projectId`:
    - You can check IDs in project settings page. Please refer to [Project Settings](/management/project/settings) documentation.

```javascript
const accessKey = process.env.DOGU_ACCESS_KEY || 'INSERT_YOUR_ACCESS_KEY';
const organizationId =
  process.env.DOGU_ORGANIZATION_ID || 'INSERT_YOUR_ORGANIZATION_ID';
const projectId = process.env.DOGU_PROJECT_ID || 'INSERT_YOUR_PROJECT_ID';
```

### Run test script

- Run the test script.

```sh
npm run test
```
