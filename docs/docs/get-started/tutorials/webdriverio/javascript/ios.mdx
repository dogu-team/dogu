:::info PREREQUISITES
You need to have an iOS device added to the device farm. Refer to [Adding Devices](/get-started/tutorials/device-farm#adding-devices) for adding an iOS device.
:::

### Clone Example Repo and Navigate to the Path

```
git clone https://github.com/dogu-team/dogu-examples.git
cd dogu-examples/webdriverio/javascript/ios
npm install
```

<Tabs groupId="target" defaultValue="chrome">
<TabItem value="chrome" label="Chrome">

### Authentication Information
* API Token
  * Check the [API Token](/management/organization/settings#api-token) on the Organization > Settings page.
* Project Id, Organization Id
  * Check the [Project Id, Organization Id](/management/project/settings#project-id) on the Organization > Project > Settings page.

<Tabs groupId="service" defaultValue="cloud">
<TabItem value="cloud" label="Cloud" default>

Apply the API Token, Project Id, and Organization Id that you obtained to **dogu-examples/webdriverio/javascript/ios/chrome.js**.

```javascript
const token = process.env.DOGU_TOKEN || 'INSERT_YOUR_TOKEN';
const organizationId = process.env.DOGU_ORGANIZATION_ID || 'INSERT_YOUR_ORGANIZATION_ID';
const projectId = process.env.DOGU_PROJECT_ID || 'INSERT_YOUR_PROJECT_ID';
```

</TabItem>
    
<TabItem value="self-hosted" label="Self-Hosted">

Apply the API Token, Project Id, and Organization Id that you obtained to **dogu-examples/webdriverio/javascript/android/chrome.js**.  
Also, apply the `DOGU_HOST` value of the configured Dogu server to apiBaseUrl.

```javascript
const token = process.env.DOGU_TOKEN || 'INSERT_YOUR_TOKEN';
const organizationId = process.env.DOGU_ORGANIZATION_ID || 'INSERT_YOUR_ORGANIZATION_ID';
const projectId = process.env.DOGU_PROJECT_ID || 'INSERT_YOUR_PROJECT_ID';
const apiBaseUrl = process.env.DOGU_API_BASE_URL || 'localhost:4000';
```

</TabItem>
</Tabs>
