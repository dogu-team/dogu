[Gamium](https://gamium.dogutech.io/) is a library for automating game testing. Can run Gamium test scripts against devices in Dogu DeviceFarm.
This document provides a pre-written Gamium test script as a sample and explains it.

:::info

If you want to know more about gamium, please refer to [Introduction](https://gamium.dogutech.io/docs/get-started/introduction).

:::
