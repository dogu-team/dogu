### Host Requirements

| OS      | OS Version                              | Architecture |
| ------- | --------------------------------------- | ------------ |
| Windows | Windows 10, Windows 11                  | x64          |
| macOS   | macOS 11 (Big Sur) or later, Xcode 12.5 | x64, arm64   |

:::caution

There may be issues with using Dogu Agent if the requirements are not met.

:::

2. In the console, click on the Host menu in the left sidebar of your organization.
   <img

   />
3. Click the Add Host button, enter a host name, and complete the creation.
4. Copy the generated host token and paste it into the token input field in the Dogu Agent to complete the connection.
   <img
     src="/img/get-started/dost-token-input.png"
     style={{ width: 600, display: 'block' }}
   />
5. Once the status of the created host becomes "Connected" in the console, the host registration is complete.
