### decoding 'command': invalid command line string

If you encounter the `error decoding 'command': invalid command line string` error, please explicitly specify the following values in the docker-compose.yml file: ${REDIS_PASSWORD}, ${TURN_SERVER_USERNAME}, ${TURN_SERVER_PASSWORD}, ${DOGU_HOST}

```
ex.
command: redis-server --requirepass ${REDIS_PASSWORD}
-> command: redis-server --requirepass dogu-onprem

user=${TURN_SERVER_USERNAME}:${TURN_SERVER_PASSWORD}, realm=${DOGU_HOST}
-> user=admin:dogu-onprem, realm=${DOGU_HOST} <- Replace it with the value you set in the environment file
```
