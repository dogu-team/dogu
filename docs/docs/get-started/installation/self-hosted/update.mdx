:::info PREREQUISITES
This document provides instructions for updating the running Dogu Self-Hosted.
:::

### Overwriting docker-compose.yml and docker-compose.env with the Latest Version

Update the content of the existing docker-compose.env file to the [latest version of docker-compose.env](https://github.com/dogu-team/dogu/blob/main/docker-compose.env).  
Update the content of the existing docker-compose.yml file to the [latest version of docker-compose.yml](https://github.com/dogu-team/dogu/blob/main/docker-compose.yml).

### Restart Dogu

```
docker ps -qa | xargs -r docker rm -f
docker compose -f ./docker-compose.yml --env-file ./docker-compose.env --verbose up -d
```
