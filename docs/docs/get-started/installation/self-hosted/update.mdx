:::info PREREQUISITES
This article describes how to update Self-Hosted from running Dogu.
:::

### 최신 버전의 docker-compose.yml, docker-compose.env 파일 생성

```
기존에 사용하던 docker-compose.env 파일을 최신 버전으로 업데이트 합니다. [docker-compose.env 최신 버전](https://github.com/dogu-team/dogu/blob/main/docker-compose.env)
기존에 사용하던 docker-compose.yml 파일을 최신 버전으로 업데이트 합니다. [docker-compose.yml 최신 버전](https://github.com/dogu-team/dogu/blob/main/docker-compose.yml)
```

### Dogu 최신 버전으로 재실행

```
docker ps -qa | xargs -r docker rm -f
docker compose -f ./docker-compose.yml --env-file ./docker-compose.env --verbose up -d
```
