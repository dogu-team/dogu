:::info PREREQUISITES
This article describes how to update Self-Hosted from a running Dogu.
:::

### Generating the Latest Version of docker-compose.yml and docker-compose.env Files

Update the existing docker-compose.env file to the latest version. [Latest version of docker-compose.env](https://github.com/dogu-team/dogu/blob/main/docker-compose.env)  
Update the existing docker-compose.yml file to the latest version. [Latest version of docker-compose.yml](https://github.com/dogu-team/dogu/blob/main/docker-compose.yml)

### Restarting Dogu with the Latest Version

```
docker ps -qa | xargs -r docker rm -f
docker compose -f ./docker-compose.yml --env-file ./docker-compose.env --verbose up -d
```
