The permissions in the organization are structured as follows:

- Organization owner
  - The user who creates the organization becomes the owner of the organization.
- `Admin`
- `Member`

## Comparing permissions

### Organization management

| Features                    | Owner | Admin | Member |
| --------------------------- | ----- | ----- | ------ |
| View organization ID        | ✅    | ✅    | ✅     |
| View API token              | ✅    | ✅    | ❌     |
| Regenerate API token        | ✅    | ✅    | ❌     |
| Remove organization         | ✅    | ❌    | ❌     |
| Change organization owner   | ✅    | ❌    | ❌     |
| Billing management          | ✅    | ❌    | ❌     |
| Change organization profile | ✅    | ❌    | ❌     |

### Project management

| Features                           | Owner | Admin | Member                           |
| ---------------------------------- | ----- | ----- | -------------------------------- |
| Create projects                    | ✅    | ✅    | ❌                               |
| Remove projects                    | ✅    | ✅    | ❌                               |
| Change project name, description   | ✅    | ✅    | Limited to participated projects |
| Add project members                | ✅    | ✅    | Limited to participated projects |
| Remove project members             | ✅    | ✅    | Limited to participated projects |
| Change project member's permission | ✅    | ✅    | Limited to participated projects |
| Assign devices to project          | ✅    | ✅    | ❌                               |
| Remove devices from project        | ✅    | ✅    | Limited to participated projects |
| View assigned devices in project   | ✅    | ✅    | Limited to participated projects |
| Create test scripts                | ✅    | ✅    | Limited to participated projects |
| Remove test scripts                | ✅    | ✅    | Limited to participated projects |
| Modify test scripts                | ✅    | ✅    | Limited to participated projects |
| View test script list              | ✅    | ✅    | Limited to participated projects |
| Execute test scripts               | ✅    | ✅    | Limited to participated projects |
| Create routines                    | ✅    | ✅    | Limited to participated projects |
| Remove routines                    | ✅    | ✅    | Limited to participated projects |
| Modify routines                    | ✅    | ✅    | Limited to participated projects |
| View routine list                  | ✅    | ✅    | Limited to participated projects |
| Run routines                       | ✅    | ✅    | Limited to participated projects |
| View project member list           | ✅    | ✅    | Limited to participated projects |
| View project list                  | ✅    | ✅    | Limited to participated projects |

:::info

The organization owner and colleagues with the `Admin` permission can view all projects.

Colleagues with the `Member` permission can only view the projects they are part of. For activity permissions within a project, please refer to the [Project Permissions](../project/permission) page.

:::
