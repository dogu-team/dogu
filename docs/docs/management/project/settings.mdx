## Project ID

<ProjectMemberPermission prefix="Retrieve project ID" />

The project ID is automatically generated when creating a project and cannot be changed.

1. Go to the project page and click on the "Settings" menu in the left sidebar.
2. Check the organization ID and project ID.
   <img
     src="/img/console/project/project-id.png"
     style={{ width: 600, display: 'block' }}
   />

## Update project

<ProjectAdminPermission prefix="Updating a project" />

1. Go to the project page and click on the "Settings" menu in the left sidebar.
2. Click on the project you want to modify.
3. Click on the "Settings" menu.
4. Update the project name and description, then click the "Save" button to complete the changes.
   <img
     src="/img/console/organization/update-project.png"
     style={{ width: 600, display: 'block' }}
   />

## Project access token {#access-token}

Project access token is used for [API](/api) and [Test Automation](/test-automation).

<ProjectAdminPermission prefix="Retrieve and regenerate project access token" />

:::danger

When regenerating project access token, the previous token will no longer be valid.

:::

1. Go to the project page and click on the "Settings" menu in the left sidebar.
2. Click on the project.
3. Click on the "Settings" menu.
4. View the existing project access token or regenerate project access token.
   <img
     src="/img/console/project/access-token.png"
     style={{ width: 600, display: 'block' }}
   />

## Change project template

<ProjectAdminPermission prefix="Change project template" />

:::danger Change project template

UI and features may vary between project templates.

:::

1. Go to the project page and click on the "Settings" menu in the left sidebar.
2. Click on the project.
3. Click on the "Settings" menu.
4. Click "Change template" button and select template for your needs.
   <img
     src="/img/console/project/project-template.png"
     style={{ width: 600, display: 'block' }}
   />

## Remove a project

<OrganizationAdminPermission prefix="Remove a project" />

:::danger Remove project

Projects contain important data such as test results. Be cautious when deleting a project as it will result in data loss.

:::

1. Go to the project page and click on the "Settings" menu in the left sidebar.
2. Click on the project you want to delete.
3. Click on the "Settings" menu.
4. Click the "Remove project" button and confirm the deletion by clicking the confirmation button.
   <img
     src="/img/console/organization/remove-project.png"
     style={{ width: 600, display: 'block' }}
   />
