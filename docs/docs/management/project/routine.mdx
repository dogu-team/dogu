:::tip

This document explains the routine functionality within a project.

For detailed information on how to create routines, concepts, and more, please refer to the [Routines](/routine/routines) document.

:::

## Create a routine

<ProjectWritePermission prefix="Creating a routine" />

1. Go to the project page and click on the "Routines" menu in the left sidebar.
2. Click the "+" button to go to the routine creation page.
3. Refer to the routine sample code or the [Routine Editor](/routine/routines/editor) document to write your routine.
   <img
     src="/img/console/project/create-routine.png"
     style={{ width: 600, display: 'block' }}
   />
4. Click the "Create" button to create the routine.

## Check Routine ID {#check-routine-id}

1. Click on the routine and check ID.
   <img
     src="/img/console/project/check-routine-id.png"
     style={{ width: 800, display: 'block' }}
   />

## Run a routine

<ProjectWritePermission prefix="Running a routine" />

:::info

For detailed information on routines, pipelines, and test execution results, please refer to the [Routines](/routine/routines) document.

:::

1. Go to the project page and click on the "Routines" menu in the left sidebar.
2. In the routine list, click on the routine you want to run.
3. Click the "Run routine" button to create a pipeline.
   <img
     src="/img/console/project/run-routine.png"
     style={{ width: 600, display: 'block' }}
   />

## Edit a routine

<ProjectWritePermission prefix="Editing a routine" />

1. Go to the project page and click on the "Routines" menu in the left sidebar.
2. In the routine list, click on the routine you want to edit.
3. Click the "Edit routine" button to go to the routine editing page.
4. Refer to the routine sample code or the [Routine Editor](/routine/routines/editor) document to make modifications to the routine.
   <img
     src="/img/console/project/edit-routine.png"
     style={{ width: 600, display: 'block' }}
   />
5. Click the "Save" button to complete the routine editing.

## Removing a routine

<ProjectAdminPermission prefix="Removing a routine" />

:::danger

Removing a routine will result in the loss of all pipeline execution results.

:::

1. Go to the project page and click on the "Routines" menu in the left sidebar.
2. In the routine list, click on the routine you want to remove.
3. Click the menu button(`···`) next to the routine title and select the "Delete routine" option.
   <img
     src="/img/console/project/delete-routine.png"
     style={{ width: 600, display: 'block' }}
   />
