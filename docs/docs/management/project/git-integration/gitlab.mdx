## Gitlab Access token Documentation

1. [Personal access token](https://docs.gitlab.com/ee/user/profile/personal_access_tokens.html)
2. [Project access token](https://docs.gitlab.com/ee/user/project/settings/project_access_tokens.html)
3. [Group access token](https://docs.gitlab.com/ee/user/group/settings/group_access_tokens.html)

## Generate Gitlab Access token

Based on self-managed, you can get a token from the path below.

### Personal access token

1. Click the top right icon
2. Click `Edit profile`
3. Click `Access Tokens`

<img
  src="/img/console/project/gitlab-personal-access-token.png"
  style={{ width: 800, display: 'block' }}
/>

### Project access token

1. Click `Project`
2. Click `Settings`
3. Click `Access Tokens`

<img
  src="/img/console/project/gitlab-project-access-token.png"
  style={{ width: 800, display: 'block' }}
/>

### Group access token

1. Click `Group`
2. Click `Settings`
3. Click `Access Tokens`

<img
  src="/img/console/project/gitlab-group-access-token.png"
  style={{ width: 800, display: 'block' }}
/>
