## Repository Access Token

1. Log in with your Bitbucket account.
2. Once you're inside the repository you want to link, navigate to `Repository settings` > `Access tokens`.
3. Click the `Create repository access token` button.
   <img
     src="/img/console/project/bitbucket-create-token.png"
     style={{ width: 500, display: 'block' }}
   />
4. Enter the token name, set the permissions, and click the `Create` button.
   :::info
   Permissions should be `Repositories` > `Read` or higher.
   :::
   <img
     src="/img/console/project/bitbucket-token-form.png"
     style={{ width: 500, display: 'block' }}
   />
5. Copy the generated token.
   <img
     src="/img/console/project/bitbucket-copy-token.png"
     style={{ width: 500, display: 'block' }}
   />
